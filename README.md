Prerequisites
=============
* Collective Knowledge framework ([@GitHub](http://github.com/ctuning/ck))
* Python 2.7 or 3.3+
* Python PIP
* Git client

Installation
============
Install CK:

```
 $ sudo pip install ck
```

Install this  CK repository:

```
 $ ck pull repo --url=https://github.com/dividiti/ck-zlib
```

Update all CK repositories at any time
```
 $ ck pull all
```
